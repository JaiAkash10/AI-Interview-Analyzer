How to Obtain Your TwelveLabs API Key and Index ID

  1. Visit the TwelveLabs website at https://twelvelabs.io.
  2. Log in to your TwelveLabs account or register for a new account.
  3. Once logged in, navigate to the API Keys section. This is usually found in your account settings or developer console.
  4. Create a new API key if you don't have one already. Copy the API key.
  5. Navigate to the Indexes section. This is typically where you manage your video indexes.
  6. Create a new index if you don't have one. Copy the Index ID.
  7. Paste the API key and Index ID into the respective fields above.
